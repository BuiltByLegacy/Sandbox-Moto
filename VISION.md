# Sandbox Moto Vision

Sandbox Moto should feel like being a kid again, building motocross tracks in the sandbox and imagining toy bikes racing them.

Every design, code, art, and systems decision should support that emotional north star.

This is not a pro racing simulator. It is not a business sim. It is a cozy motocross imagination toy: warm sand, handmade track shapes, tiny toy dirt bikes, oversized backyard props, and playful race stories.

The core loop is simple:

Build track -> run toy race -> get imagination feedback -> modify track -> race again.
