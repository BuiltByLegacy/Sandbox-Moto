# Toy Bike Reference

Toy bikes should feel like small play objects:

- Bright colors.
- Simple readable shapes.
- Chunky tires.
- Small number plates.
- Slightly exaggerated proportions.
- Minimal mechanical detail.

They should not look like hyper-real motocross bikes. The charm comes from toy scale and imagination.

Rider colors can help feedback:

- Blue bike.
- Red bike.
- Green bike.
- Yellow bike.
- Purple bike.

Future animation notes:

- Tiny wobble.
- Small hop arcs.
- Gentle spinout crashes.
- Playful roost puffs.
