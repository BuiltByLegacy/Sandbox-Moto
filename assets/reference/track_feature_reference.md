# Track Feature Reference

Track features should be readable from a distance and feel handmade from sand or dirt.

## Features

- Single: one small mound.
- Double: two separated mounds.
- Triple: three larger rhythm mounds.
- Tabletop: broad flat-topped mound.
- Whoops: tight repeated bumps.
- Rollers: smoother rounded bumps.
- Sand: soft patch with lighter grain.
- Hill: larger smooth elevation mound.

## Visual Notes

- Use warm dirt tones against golden sand.
- Keep silhouettes clear.
- Favor soft rounded forms.
- Make features feel sculpted by a child's hands.

## Gameplay Notes

- Beginner riders roll or crash on bigger features.
- Better riders clear bigger features.
- Sand slows weak sand riders.
- Whoops punish low whoop skill.
