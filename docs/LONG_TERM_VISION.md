# Long-Term Vision

## Product Path

Version 1: Cozy sandbox prototype.

Version 2: Better sculpting and toy bikes.

Version 3: Scrapbook and toy box progression.

Version 4: Multiple memory locations.

Version 5: Track sharing / Steam Workshop-style community.

Version 6: Sandbox neighborhood / larger world.

## Release Strategy

Launch on Steam first.

PC is the best first platform because mouse input and screen space fit creative building.

iPad later may be excellent for touch sandbox play.

iPhone later should be simplified for smaller sessions and smaller interactions.
