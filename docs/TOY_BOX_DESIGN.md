# Toy Box Design

Menus should feel like opening a toy box, not inventory management.

## Toy Box Contents

- Bikes.
- Riders.
- Shovels.
- Buckets.
- Rakes.
- Cones.
- Flags.
- Toy trucks.
- Dinosaurs.
- Army men.
- Fences.
- Decorations.
- Track molds.
- Toy spectators.
- Old tires.
- Plastic tunnels.
- Cardboard ramps.
- Stickers and number plates.

## Menu Tone

The player should feel like they found another toy to play with, not like they optimized a loadout.
