# Emotional Pillars

## Nostalgia

Features should recall afternoons spent making tracks from dirt, sand, carpet, snow, or piles behind the garage. Use props, sound, camera, and scale to imply childhood without saying it directly.

## Creativity

The player should feel free to build odd, funny, ambitious, and imperfect tracks. Tools should reward experimenting, rebuilding, and trying one more layout.

## Imagination

Toy bikes come alive when Play is pressed. Riders, feedback, scrapbook memories, and toy crowds should feel imagined rather than simulated.

## Discovery

Progression should feel like finding new toys in a toy box: new bike colors, props, molds, stickers, tracks, worlds, and scrapbook frames.

## Relaxation

No pressure, no game over, no financial stress. The player can shape sand, listen to ambience, and run races because it feels good.

## Pride

The player should feel proud when a section finally flows, one tiny rider clears the impossible jump, or a favorite track earns a scrapbook memory.

## Storytelling

Races should create little stories through feedback bubbles, temporary rider personalities, crashes, near-misses, and favorite toy-bike moments.
