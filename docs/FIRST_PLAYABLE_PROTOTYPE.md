# First Playable Prototype

This is the minimum working Sandbox Moto prototype target.

## Must Prove

- The player can enter Play Time and build a simple smooth track.
- The player can place a start gate, finish marker, and readable obstacle placeholders.
- The player can press Play Race.
- The sandbox comes alive with multiple toy bikes.
- Toy bikes follow the track from the start area toward the finish.
- Toy bikes react to jumps, whoops, sand, rollers, hills, and berm placeholders.
- Imagination feedback appears after the race.
- The game returns to Play Time.
- The player can run one more race.

## Out Of Scope

- Full terrain simulation.
- Toy box progression.
- Memory Book implementation.
- Steam features.
- Steam Workshop.
- Multiple memory worlds.
- Advanced replay or photo mode.
