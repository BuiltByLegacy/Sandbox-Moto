# Play Time Design

## Goal

Play Time is the heart of Sandbox Moto. It should feel like shaping a sandbox track with imagination, not operating a professional track editor.

## Core Actions

- Draw a smooth track path.
- Place a start gate.
- Place a finish marker.
- Add jumps and terrain features.
- Adjust the track after watching races.

## Interaction Style

The player should be able to build quickly and casually:

- Simple tool panel.
- Clear tool names and icons.
- Smooth drawing instead of blocky grid placement.
- Immediate visual feedback.
- Easy iteration.

The current prototype should preserve free-draw smooth track behavior. Do not replace it with a blocky grid system unless the design intentionally shifts.

## Future Build Tools

Potential tools:

- Smooth track brush.
- Widen/narrow track brush.
- Raise/lower mound brush.
- Berm brush.
- Eraser.
- Move obstacle.
- Rotate obstacle.
- Duplicate obstacle.
- Decorative prop placement.

## Tone

Play Time should feel calm and creative. There is no timer, score pressure, economy, or failure state.
