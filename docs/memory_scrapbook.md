# Memory Scrapbook

## Goal

The Memory Scrapbook saves favorite imagined race moments like a kid taping a Polaroid into a notebook.

It should feel cozy and sentimental, not like a statistics archive.

## Saveable Memory Fields

Each memorable race can save:

- Track name.
- Winning bike color.
- Best jump.
- Biggest crash.
- Funniest feedback bubble.
- Small snapshot or Polaroid-style image.

## Kid-Style Track Names

Implemented: `scripts/TrackNamer.gd` generates these, the nameplate above the sandbox shows and edits them, and they persist with the save.

Generated track names should sound like a kid made them:

- Dragon Hill.
- Mega Jump Raceway.
- Backyard National.
- Big Sand SX.
- The Impossible Triple.
- Bucket Turn Speedway.
- Dad's Garage Supercross.

## One More Race

After every race, show a soft prompt:

One more race?

This should feel like childhood play, not a competitive retry screen. It is an invitation to keep imagining, fix one thing, and watch the toys come alive again.

## Mom Called Dinner Pause / Save

The "Mom called Dinner" moment is a cozy pause/save concept:

- The world freezes.
- Sunlight fades slightly.
- Tiny engines go quiet.
- Dust hangs for a moment.
- The sandbox is saved exactly how the player left it.

This should feel like playtime being paused, not failure or interruption.
