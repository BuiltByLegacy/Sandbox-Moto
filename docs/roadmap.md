# Roadmap

## Phase 1: Playable Prototype

- Smooth free-draw track.
- Start and finish placement.
- Basic obstacle placement.
- Toy riders follow the track.
- Random rider identities and skills.
- Basic obstacle outcomes.
- Imagination feedback bubbles.

## Phase 2: Better Building

- Move, rotate, and delete obstacles.
- Add track width controls.
- Add berms.
- Add simple undo/redo.
- Add save/load for sandbox tracks.
- Improve tool panel with icons and bubbles.

## Phase 3: Richer Toy Racing

- Better lane behavior.
- More expressive toy bike animation.
- Tiny roost puffs.
- Cleaner jump arcs.
- Replay moments.
- More varied rider personalities without tying names to skills.

## Phase 4: Cozy Diorama Art

- Handmade sand and dirt materials.
- Toy bike art.
- Buckets, shovels, cones, grass blades, fences, and pits.
- Warm lighting.
- Camera polish.

## Phase 5: Sharing And Expansion

- Track sharing.
- Screenshot/postcard mode.
- More obstacle packs.
- Decorative prop packs.
- Seasonal sandbox themes.
