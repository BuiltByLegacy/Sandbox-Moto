# North Star

If someone who grew up riding or loving motocross smiles because Sandbox Moto reminds them of spending entire afternoons building tracks in the dirt with toy bikes, we succeeded.

Every mechanic, asset, tool, sound, UI choice, camera move, and progression system should answer:

Does this make the player feel more like they are reliving a childhood motocross sandbox memory?

## Design Filter

Use this filter before adding a feature:

- Does it increase nostalgia, creativity, imagination, or tactile play?
- Does it make the world feel tiny, warm, handmade, playful, or emotionally familiar?
- Does it support build -> race -> feedback -> adjust -> one more race?
- Does it avoid turning the game into a simulator, manager, business game, or pro racing fantasy?
- Does it unlock imagination rather than power?

If the answer is no, simplify or cut the idea.
