# Progression

## Goal

Progression in Sandbox Moto should feel like finding new toys, remembering favorite tracks, and expanding the toy box.

It should not be money-first. Avoid business, sponsor, promoter, factory-team, or career-management framing.

## Toy Box Unlocks

Unlocks should feel like a kid discovering more things to build with:

- Bike colors.
- Toy riders.
- Cones.
- Flags.
- Old tires.
- Popsicle-stick fences.
- Toy dump trucks.
- Plastic tunnels.
- Cardboard ramps.
- Toy spectators.
- Stickers and number plates.
- Shovels.
- Buckets.
- Rakes.
- Molds.
- Mini excavators.
- Bridges.
- PVC pipes.
- Toy trees.
- Toy campers.

## Unlock Tone

Use playful wording:

- Found a new toy bike.
- Added stickers to the toy box.
- Found old tires for track markers.
- Added a cardboard ramp.
- Saved a favorite race in the scrapbook.

Avoid:

- Earned prize money.
- Signed sponsors.
- Increased revenue.
- Managed attendance.
- Bought a factory upgrade.

## Collection Use

The collection should feed Play Time:

- More ways to decorate tracks.
- More toy-bike color identities.
- More props for memory worlds.
- More visual variety for fake Polaroids and scrapbook pages.
