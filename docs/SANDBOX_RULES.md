# Sandbox Rules

The sandbox is the main toy. Dirt and sand should be tactile, forgiving, and rebuildable.

## Core Behaviors

- Pile dirt.
- Smooth dirt.
- Flatten.
- Carve ruts.
- Build berms.
- Shape jumps.
- Water sections.
- Repair jumps.
- Knock down and rebuild.

## Played-In Track

The track can look played in after races:

- Darker ruts.
- Tire marks.
- Roost piles.
- Worn jump faces.
- Sand pushed out of corners.
- Smoother landings after repeated races.

Played-in dirt should feel like a memory of the races that happened there, not a punishment.
